Hvad er Javascript?

JavaScript blev oprindeligt skabt for at "gøre websider levende". Programmer i dette sprog kaldes scripts. De kan skrives direkte i en websides HTML og køres automatisk, når siden indlæses. Scripts leveres og udføres som almindelig tekst. De behøver ikke en speciel forberedelse eller kompilering for at køre. På denne måde adskiller JavaScript sig fra et andet sprog kaldet Java.

Javascript Funktion

En JavaScript-funktion er en blok af kode, der er designet til at udføre en bestemt opgave. Funktioner kan tage input (kaldet parametre eller argumenter), udføre handlinger baseret på dette input og returnere et resultat.
