som følger samme mønster som de eksisterende klasser Helmet og Chestplate

Løsning og kodeforklaring

interface Armor {
void equip();
}

Opretter et interface Armor med en metode equip().
Alle rustningsgenstande (fx hjelm og brystplade) skal implementere denne metode.

class ArmorSet implements Armor {
ArrayList armors = new ArrayList();
void addArmor(Armor armor) {
armors.add(armor);
}
public void equip() {
for (Armor armor : armors) {
armor.equip();
}
}
}

ArmorSet er en klasse, der implementerer Armor-interfacet.
Dette betyder, at ArmorSet skal implementere metoden equip(), da den er defineret i Armor-interfacet.
armors er en liste (ArrayList), der kan indeholde objekter af typen Armor.
Dette gør det muligt at tilføje forskellige rustningsdele (f.eks. Helmet, Chestplate) til sættet.
Metode til at Tilføje Rustningsdele
addArmor(Armor armor):
En metode, der tager et objekt af typen Armor som parameter.
Tilføjer det til listen armors.
Dette gør det muligt at bygge et sæt af rustningsdele dynamisk.
Implementering af equip()-Metoden :
equip()-metoden kaldes for at tage hele rustningssættet på.
Den gennemløber (for-each loop) listen armors og kalder equip() på hver enkelt rustningsdel.
Da Helmet og Chestplate (eller andre Armor-typer)
implementerer equip(), vil de hver især udføre deres specifikke equip()-handling.
